# Sinhala Quran

A comprehensive Al-Quran application for Sinhala speakers, built with Flutter.

## Features

- **Multiple Translations**: Access Quran translations in Sinhala, including PJ and TNTJ.
- **Audio Playback**: Listen to Quran recitations with synchronized verse highlighting.
- **Verse Options**:
  - Copy Arabic or Translation text
  - Share Verses directly
  - Bookmark your favorite or last read verses
- **Customization**:
  - **Dark Mode**: Fully supported dark theme for comfortable night reading.
  - **Font Settings**: Choose from multiple Arabic and Sinhala fonts.
  - **Text Size**: Adjustable font sizes for both Arabic and translation text.
- **Reciter Selection**: Choose from available reciters.

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- Flutter SDK `^3.0.0` (or compatible version as per `pubspec.yaml`)
- Dart SDK

### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/Start-app-Auth/sinhala_quran.git
   ```
2. Install dependencies
   ```sh
   flutter pub get
   ```
3. Run the app
   ```sh
   flutter run
   ```
